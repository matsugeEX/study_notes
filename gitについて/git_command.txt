・git remote add <α> <β>

α-任意の名前を設定できるが origin が慣例。git clone の際にも自動的に origin が設定される。

β-接続先のリモートリポジトリのURL。<例> https://github.com/username/example.git


・git remote -v

接続しているリモートリポジトリの確認。


・git remote remove <α>

α- git remote -v で確認した、先頭のリモート名。URLではない。　<例> git remote remove origin


・git remote set-url <α> <β>

α- git remote -v で確認した、先頭のリモート名。URLではない。

β-接続先のリモートリポジトリのURL。<例> https://github.com/username/new_example.git


・git branch -a

ブランチの一覧を見る。


・git checkout -b <α>

α-作成するブランチの名前。ローカルでブランチを作成する。


・git push -u origin <α>

α-作成したブランチ名。ブランチをリモートに登録。
